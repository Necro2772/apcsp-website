Linux Commands

Command Description Example Usage
echo Echo strings to standard output echo file1
rm Removes the file rm file
ls Lists contents of directory ls
cp Copy source file to destination cp file1 directory1
man Provides information on a command man man
cat Writes out file to output cat file
more Writes out file on screen at a time more file
less Writes out file in scrollable format less file
uname Print system information uname
mkdir Creates directory mkdir directory
